# Explore Egypt 🇪🇬

A modern tourism discovery platform built with React that helps users explore Egypt's most famous attractions, historical landmarks, beaches, and hidden gems.

## Features

* Search tourist destinations
* Filter places by city
* View detailed information about attractions
* Add destinations to Favorites
* Dark Mode support
* Responsive design
* Interactive Google Maps links

## Technologies Used

* React.js
* React Router
* Context API
* Bootstrap
* CSS
* Local Storage

## Project Overview

Explore Egypt was created to simplify tourism discovery in Egypt by providing a clean and user-friendly platform where travelers can easily find and save destinations of interest.

The platform focuses on improving the travel planning experience through modern design and simple navigation.
